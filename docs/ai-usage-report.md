# AI Usage Report

## Tools Used & Use Cases

### 1. ChatGPT
I used ChatGPT as a development assistant throughout the project. It supported me in multiple areas:

- **JavaScript Development**
  - Helped implement complex logic such as project filtering, sorting.
  - Helped integrate an external API (quotes API).

- **API Integration**
  - Guided the process of fetching data from a public API (DummyJSON quotes API).
  - Helped implement automatic quote cycling every 5 seconds.

- **State Management**
  - Assisted in using `localStorage` to store user preferences such as:
    - Dark mode theme
    - Visitor name
  - Helped implement logic to retrieve and update stored data.


- **Debugging**
  - Helped identify and fix logic errors in:
    - Form validation
    - Project filtering
---

## Benefits and Challenges

### Benefits
- Fasters development and implementation of features.
- Improved code quality and structure.
- Better understanding of JavaScript concepts like:
  - Asynchronous programming (fetch API).

### Challenges
- Some AI-generated code needed adjustments to fit my existing structure.
- CSS suggestions sometimes conflicted with existing styles and required cleanup.
- Required careful testing to ensure all features worked correctly together.

---

## Learning Outcomes

Through using AI tools, I learned:

- How to integrate and use external APIs in a web application.
- How to manage application state using `localStorage`.
- How to debug and refine AI-generated code instead of using it directly.

---

## Responsible Use and Modifications

I ensured responsible use of AI by reviewing and modifying all generated content:

- Tested every feature in the browser before finalizing it.
- Adjusted and simplified the AI generated code to improve readability.
- Modified UI and styling to match my own design preferences.
- Ensured the final implementation reflects my own understanding and work.
- Combined AI suggestions with my own logic and improvements.
