# Professional Portfolio Website

## Project Description

This project is a responsive personal portfolio website built using HTML, CSS, and JavaScript.  

The website includes three main sections:

- **About Me** – A short introduction, tagline, and profile image.
- **Projects** – A showcase of selected projects with descriptions and external links.
- **Contact** – A contact form with Name, Email, and Message fields (front-end only).

The portfolio is fully responsive and includes a **dark mode toggle feature** for improved user experience. The design focuses on clean layout, accessibility, and modern UI styling.

---

## Setup Instructions (Run Locally)

To run this project locally:

1. Download or clone the repository.
2. Open `index.html` in any modern web browser:
   - Right-click → Open with → Browser
